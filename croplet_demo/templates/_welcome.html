Welkom op de Crop-R Demo applicatie

Deze demo applicatie maakt gebruik van de nieuwe Crop-R API met oauth2 ondersteuning. Om gebruik te maken van deze applicatie moet je eerst een account aanmaken in de database. Ga dan naar Rainfall en druk op autoriseren.

De demo applicatie haalt alle gewaspercelen op van de Crop-R account en koppelt de actuele neerslagvoorspelling voor de komende 2 uur op. Hiervoor wordt de buienradar api gebruikt.
